## このプロジェクトについて

『失敗者の告白』は、AI（LLM）を活用して執筆されたビジネスナラティブ小説です。

本プロジェクトでは、以下の2つを同時に実現することを目指しています：

1. **AI を活用した小説の作り方を、実例とともに公開する**
2. **読者コミュニティの力で、作品をさらに磨き上げる**

小説の原稿はすべて Markdown で管理されており、誰でも閲覧・フィードバック・改善提案が可能です。

## 作品概要

人材紹介会社に転職した主人公。  
元起業家のコーチ・佐伯零との出会いをきっかけに、「ビジネス」と「自分」の本質に向き合う物語——。

### 構成

| 部 | テーマ | 章 |
|:---|:---|:---|
| **第1部** 自分という壁 | プレイヤー | 第一章〜第四章 |
| **第2部** 他者という壁 | マネージャー | 第五章〜第八章 |
| **第3部** 世界という壁と統合 | リーダー | 第九章〜第十二章 |

## AI 協働創作について

本作は、LLM（大規模言語モデル）を「共著者」ではなく「設計図に基づくビルドエンジン」として活用する手法で制作されています。

- **設計駆動**: 人間が物語の構造・キャラクター・美学を設計し、AI が実装する
- **反復改善**: v1〜v23 に至る23回以上のイテレーションを経て現在の形に到達
- **自律推敲**: AI による3パス自動推敲（美学検閲・感覚注入・整合性監査）

この制作プロセス自体が、AI 時代の新しい創作の形を探る実験です。
