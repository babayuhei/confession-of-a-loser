# 行動規範（Code of Conduct）

## 私たちの約束

このプロジェクトへの参加者全員にとって、安全で包括的な環境を提供するため、  
私たち（管理者・コントリビューター双方）は以下の行動規範を定めます。

## 歓迎する行動

- 丁寧で建設的なフィードバック
- 異なる意見や経験を尊重すること
- 批判は人ではなく作品・コードに向けること
- 新しい参加者を歓迎し、サポートすること
- 感謝と敬意を示すこと

## 受け入れられない行動

- 個人攻撃・侮辱・ハラスメント
- 政治的・宗教的な攻撃的言動
- 性的なコンテンツやハラスメント
- 意図的な荒らし行為
- プライバシー情報の無断公開

## 適用範囲

この行動規範は、GitHub Issues・Pull Requests・Discussions 等、  
このプロジェクトに関連するすべての公開の場に適用されます。

## 執行

問題のある行動を発見した場合は、Issue または次の連絡先にご報告ください：

- GitHub Issues にて `[報告]` タグをつけて起票してください
- 深刻な問題の場合は、リポジトリオーナー（馬場祐平）に直接連絡してください

管理者は状況を確認の上、コメントの削除・Issue のクローズ・参加制限等の対応を取ることができます。

## 帰属

本行動規範は [Contributor Covenant](https://www.contributor-covenant.org) v2.1 を参考に作成しました。
